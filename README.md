# Typing Tutor (Day 9)

Offline-first typing speed trainer built to improve basic computer literacy.
Tracks **WPM**, **accuracy**, and **time** using a simple browser-based interface.

Part of a 90-day challenge to build practical solutions for Africa.

---

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

No frameworks. No backend.

---

## Features
- Live typing feedback
- Words Per Minute (WPM) tracking
- Accuracy calculation
- Session timer
- Reset / retry support
- Works fully offline
